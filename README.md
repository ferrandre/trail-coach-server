# Trail Coach Server

Proxy server per Trail Coach — gestisce le chiamate Strava API.

## Deploy su Render

1. Crea un account su render.com
2. New > Web Service > collega questo repository GitHub
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Piano: Free

Il server sarà disponibile su `https://trail-coach-server.onrender.com`
